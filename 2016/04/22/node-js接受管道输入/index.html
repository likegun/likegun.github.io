Node.js接受管道输出

场景:

Request

Response

{"name":"wqr","sex":"male","age":22}

目标:

调用curl获取相应之后，格式化得输出JSON

方法:

| 符号是linux系统下的管道符号,管道符号能将左边的命令的输出当做右边命令的输入，而json命令我在~/.bash_profile定义的node相关命令的别名别名。

在~/.bash_profile中添加:

alias json='node -e "var s=process.openStdin(); var d=[]; s.on(\"data\",function(c){d.push(c);});s.on(\"end\",function(){console.log(JSON.stringify(JSON.parse(d.join('')),null,2));});"'

这句话声明json是下面这条长命令的别名

node -e "var s=process.openStdin(); var d=[]; s.on(\"data\",function(c){d.push(c);});s.on(\"end\",function(){console.log(JSON.stringify(JSON.parse(d.join('')),null,2));});"'

这条命令用node -e去执行一段Node.js脚本
